Web Browsers Definition List

Internet Explorer
Developed by Microsoft, an integral piece of Windows products.
Mozilla
Developed by the Mozilla Project, an open-source browser for multiple platforms.
Netscape
Developed by Netscape Communications Corporation, one of the first graphical browsers.
Safari
Developed by Apple Computer, Inc., for Apple's OS X operating system.
Firefox
A "next-generation" open-source browser developed by Mozilla and available on multiple platforms.
